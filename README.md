# BCS601-USN4MW23CS103

## Project Title
Cloud Based Java Project

## Description
This project demonstrates basic Java programming concepts. 
It calculates factorial values from 4 to 8 and finds minimum and maximum values from a given array.

## Technologies Used
- Java
- GitHub (Cloud Platform)

## Steps to Run
1. Compile the program:
   javac Main.java
2. Run the program:
   java Main

## Output
4 24  
5 120  
6 720  
7 5040  
8 40320  
min = 20  
max = 44  

## Deployment
The project is deployed using GitHub cloud platform.
