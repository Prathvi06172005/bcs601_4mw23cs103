# BCS601-USN4MW23CS103

## Project Title
Cloud Based Java Project

## Description
This project demonstrates basic Java programming concepts. It calculates factorial values and finds minimum and maximum values from given inputs.

## Technologies Used
- Java
- HTML, CSS, JavaScript
- Netlify (Cloud Platform)
- GitHub

## Project Files
- Main.java (Java implementation)
- index.html (Web-based version for deployment)

## Steps to Run (Java)
1. Compile the program:
   javac Main.java  
2. Run the program:
   java Main  

## Output
4! = 24  
5! = 120  
6! = 720  
7! = 5040  
8! = 40320  
Min = 20  
Max = 44
